## For the following tasks:
- Need:
    - pipenv: `pip install pipenv`
    - Docker
    - Make
- Make sure you're in the root project directory
- To start:
```
$ pipenv install --dev
...
$ pipenv shell
...
```

# Training:
```
$ pipenv run train (optional args)
```

# Inference:
```
$ pipenv run eval (Number of output tokens)
```
